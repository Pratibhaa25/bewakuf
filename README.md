# Pratibha Pf — Portfolio

This repository contains a minimal responsive portfolio website.

## Structure

- `index.html` — main HTML file
- `styles.css` — site styles
- `script.js` — simple JavaScript interactions
- `images/profile.svg` — placeholder profile image

## Local preview

To preview the site locally (requires Python 3 installed):

```bash
cd "Pratibha Pf"
python3 -m http.server 8000
# Open http://localhost:8000 in a browser
```

Alternatively, use VS Code Live Server extension and open `index.html`.

## Customization

- Replace `images/profile.svg` with your own photo or SVG.
- Edit copy and projects in `index.html`.
- Update colors in `styles.css`.

## Deploy

You can deploy this static site to GitHub Pages, Netlify, Vercel, or any static site host. For GitHub Pages, push the repository to GitHub and enable Pages from the repository settings.

Enjoy!
